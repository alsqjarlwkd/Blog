# Blog Site
### REST API CRUD 를 이용한 블로그 제작

### 사용스택
>REACT,JSON,JSON_SERVER,Fetch,REST API,CUSTOM HOOK,REACT ROUTER(Hooks:UseParams,UseHistory)

### 환경세팅
서버와 연동을 위한 json-server 실행
>json-server --watch ./src/DB/Data.json --port 3001

서버 실행 후 node 실행
>npm start
